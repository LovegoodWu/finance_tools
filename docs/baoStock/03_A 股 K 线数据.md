# A 股 K 线数据

## 获取历史 A 股 K 线数据：query_history_k_data_plus()

### 方法说明
通过 API 接口获取 A 股历史交易数据，可以通过参数设置获取日 k 线、周 k 线、月 k 线，以及 5 分钟、15 分钟、30 分钟和 60 分钟 k 线数据。

- **返回类型：** pandas 的 DataFrame 类型
- **数据范围：** 1990-12-19 至今
- **复权类型：** 不复权、前复权、后复权

---

## 使用示例

### 日线数据
```python
import baostock as bs
import pandas as pd

lg = bs.login()
rs = bs.query_history_k_data_plus("sh.600000",
    "date,code,open,high,low,close,preclose,volume,amount,adjustflag,turn,tradestatus,pctChg,isST",
    start_date='2024-07-01', end_date='2024-12-31',
    frequency="d", adjustflag="3")
data_list = []
while (rs.error_code == '0') & rs.next():
    data_list.append(rs.get_row_data())
result = pd.DataFrame(data_list, columns=rs.fields)
result.to_csv("history_A_stock_k_data.csv", index=False)
bs.logout()
```

### 分钟线数据
```python
rs = bs.query_history_k_data_plus("sh.600000",
    "date,time,code,open,high,low,close,volume,amount,adjustflag",
    start_date='2024-07-01', end_date='2024-12-31',
    frequency="5", adjustflag="3")
```

---

## 参数说明

| 参数 | 说明 |
|------|------|
| code | 股票代码，sh 或 sz.+6 位数字，如 sh.601398 |
| fields | 指标字段，逗号分隔 |
| start | 开始日期 YYYY-MM-DD，默认 2015-01-01 |
| end | 结束日期 YYYY-MM-DD，默认最近交易日 |
| frequency | d=日，w=周，m=月，5/15/30/60=分钟线 |
| adjustflag | 3=不复权，1=后复权，2=前复权 |

---

## 返回字段

| 字段 | 说明 |
|------|------|
| date | 交易所行情日期 |
| code | 证券代码 |
| open/high/low/close | 开/高/低/收盘价 |
| preclose | 前收盘价 |
| volume/amount | 成交量/成交额 |
| adjustflag | 复权状态 |
| turn | 换手率 |
| tradestatus | 交易状态 (1 正常 0 停牌) |
| pctChg | 涨跌幅 |
| peTTM/pbMRQ/psTTM/pcfNcfTTM | 估值指标 |
| isST | 是否 ST |
