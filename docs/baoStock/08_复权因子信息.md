# 复权因子信息

## query_adjust_factor()

### 方法说明
获取复权因子信息数据。

**注意：** BaoStock 使用**涨跌幅复权算法**

---

## 使用示例

```python
import baostock as bs
import pandas as pd

lg = bs.login()
rs_list = []
rs = bs.query_adjust_factor(code="sh.600000", 
    start_date="2015-01-01", end_date="2017-12-31")
while (rs.error_code == '0') & rs.next():
    rs_list.append(rs.get_row_data())
result = pd.DataFrame(rs_list, columns=rs.fields)
result.to_csv("adjust_factor.csv", index=False)
bs.logout()
```

---

## 参数说明

| 参数 | 说明 |
|------|------|
| code | 股票代码 |
| start_date | 开始日期，默认 2015-01-01 |
| end_date | 结束日期，默认当前日期 |

---

## 返回字段

| 字段 | 说明 | 计算说明 |
|------|------|----------|
| code | 证券代码 | - |
| dividOperateDate | 除权除息日期 | - |
| foreAdjustFactor | 向前复权因子 | 前收盘价/除权日前收盘 |
| backAdjustFactor | 向后复权因子 | 除权日前收盘/前收盘 |
| adjustFactor | 本次复权因子 | - |

---

## 复权价格计算

**前复权：** 复权价格 = 原始价格 × foreAdjustFactor

**后复权：** 复权价格 = 原始价格 × backAdjustFactor
