# 估值指标 (日频)

## 通过 query_history_k_data_plus() 获取

### 说明
获取沪深 A 股估值指标 (日频) 数据。

**注意：** 
- 指数未提供估值数据
- 未提供周、月估值数据

---

## 使用示例

```python
import baostock as bs
import pandas as pd

lg = bs.login()
# 获取估值指标
rs = bs.query_history_k_data_plus("sh.600000",
    "date,code,close,peTTM,pbMRQ,psTTM,pcfNcfTTM",
    start_date='2015-01-01', end_date='2017-12-31', 
    frequency="d", adjustflag="3")

data_list = []
while (rs.error_code == '0') & rs.next():
    data_list.append(rs.get_row_data())
result = pd.DataFrame(data_list, columns=rs.fields)
result.to_csv("valuation_data.csv", index=False)
bs.logout()
```

---

## 估值指标说明

| 指标 | 说明 | 计算公式 |
|------|------|----------|
| peTTM | 滚动市盈率 | 收盘价/每股盈余 TTM |
| pbMRQ | 市净率 | 收盘价/每股净资产 |
| psTTM | 滚动市销率 | 收盘价/每股销售额 |
| pcfNcfTTM | 滚动市现率 | 收盘价/每股现金流 TTM |

---

## 返回字段

| 字段 | 说明 |
|------|------|
| date | 交易所行情日期 |
| code | 证券代码 |
| close | 收盘价 |
| peTTM | 滚动市盈率 |
| pbMRQ | 市净率 |
| psTTM | 滚动市销率 |
| pcfNcfTTM | 滚动市现率 |
