# 证券基本资料

## query_stock_basic()

### 方法说明
获取证券基本资料信息。

- **返回类型：** pandas DataFrame

---

## 使用示例

```python
import baostock as bs
import pandas as pd

lg = bs.login()
# 按代码查询
rs = bs.query_stock_basic(code="sh.600000")
# 按名称模糊查询
# rs = bs.query_stock_basic(code_name="浦发银行")

data_list = []
while (rs.error_code == '0') & rs.next():
    data_list.append(rs.get_row_data())
result = pd.DataFrame(data_list, columns=rs.fields)
result.to_csv("stock_basic.csv", index=False)
bs.logout()
```

---

## 参数说明

| 参数 | 说明 |
|------|------|
| code | A 股股票代码，可为空 |
| code_name | 股票名称，支持模糊查询，可为空 |

**注意：** 参数为空时返回全部股票基本信息

---

## 返回字段

| 字段 | 说明 |
|------|------|
| code | 证券代码 |
| code_name | 证券名称 |
| ipoDate | 上市日期 |
| outDate | 退市日期 |
| type | 证券类型 (1 股票 2 指数 3 其它 4 可转债 5ETF) |
| status | 上市状态 (1 上市 0 退市) |
