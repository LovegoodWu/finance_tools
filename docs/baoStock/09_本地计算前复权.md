# 本地计算前复权

## 利用复权因子手动计算复权价格

### 说明
为避免频繁请求 API，可通过复权因子与本地 K 线数据自行计算复权价格。

---

## 完整示例

```python
import baostock as bs
import pandas as pd

lg = bs.login()
stock_code = "sh.600000"

# 1. 读取本地 K 线数据
kline_df = pd.read_csv("sh600000_kline.csv")
kline_df['date'] = pd.to_datetime(kline_df['date'])

# 2. 获取复权因子数据
factor_df = bs.query_adjust_factor(stock_code, 
    start_date="2010-01-01", end_date="2020-01-01").get_data()
factor_df['dividOperateDate'] = pd.to_datetime(factor_df['dividOperateDate'])
factor_df['foreAdjustFactor'] = factor_df['foreAdjustFactor'].astype(float)

# 3. 为每个交易日匹配复权因子
def get_factor(trade_date):
    mask = factor_df['dividOperateDate'] <= trade_date
    if mask.any():
        return factor_df.loc[mask, 'foreAdjustFactor'].iloc[-1]
    return 1.0

kline_df['adj_factor'] = kline_df['date'].apply(get_factor)

# 4. 计算前复权价格
kline_df["adj_open"] = kline_df["open"] * kline_df["adj_factor"]
kline_df["adj_high"] = kline_df["high"] * kline_df["adj_factor"]
kline_df["adj_low"] = kline_df["low"] * kline_df["adj_factor"]
kline_df["adj_close"] = kline_df["close"] * kline_df["adj_factor"]

kline_df.to_csv("sh600000_adjusted.csv", index=False)
bs.logout()
```

---

## 关键要点

1. **复权因子获取范围：** 需要获取比 K 线数据更早的复权因子
2. **复权因子匹配：** 使用小于等于交易日期的最接近复权因子
3. **计算公式：** 复权价格 = 原始价格 × 复权因子
