# 黑名单

## 黑名单访问统计

**重要提示：** 每日 API 请求不能超过 10 万次，超过后进入黑名单控制。

**黑名单控制后：** 请在 QQ 群中联系管理员并告知互联网 IP 地址。

---

## 示例代码：查看 IP 是否加入黑名单

```python
import baostock as bs

# 示例代码查看 IP 是否加入黑名单
lg = bs.login()

# 显示登陆返回信息
print('login respond error_code:'+lg.error_code)
print('login respond error_msg:'+lg.error_msg)

if(lg.error_code == "10001011"):
    print("IP 已经加入黑名单，需要去 QQ 群里求助")
```

---

## 黑名单说明

BaoStock 对 API 访问频率进行了限制：
- **每日请求限制：** 10 万次/天
- **超限后果：** IP 地址被加入黑名单
- **解决方法：** 在 QQ 群中联系管理员，提供互联网 IP 地址申请解封

---

## 使用建议

1. **合理控制请求频率：** 避免短时间内大量请求
2. **本地缓存数据：** 将获取的数据保存到本地，减少重复请求
3. **批量获取数据：** 尽量一次性获取所需的全部数据
4. **使用复权因子：** 通过本地计算复权价格，减少 API 调用
