# 证券元信息

## 1. 查询交易日：query_trade_dates()

### 使用示例
```python
import baostock as bs
import pandas as pd

lg = bs.login()
rs = bs.query_trade_dates(start_date="2017-01-01", end_date="2017-06-30")
data_list = []
while (rs.error_code == '0') & rs.next():
    data_list.append(rs.get_row_data())
result = pd.DataFrame(data_list, columns=rs.fields)
result.to_csv("trade_dates.csv", index=False)
bs.logout()
```

### 返回字段
| 字段 | 说明 |
|------|------|
| calendar_date | 日期 |
| is_trading_day | 是否交易日 (0 非交易日 1 交易日) |

## 2. 查询证券列表：query_all_stock()

### 使用示例
```python
import baostock as bs
import pandas as pd

lg = bs.login()
# 查询指定日期的所有证券
rs = bs.query_all_stock(day="2024-10-25")
data_list = []
while (rs.error_code == '0') & rs.next():
    data_list.append(rs.get_row_data())
result = pd.DataFrame(data_list, columns=rs.fields)
result.to_csv("all_stock.csv", index=False)
bs.logout()
```

### 参数说明
| 参数 | 说明 |
|------|------|
| day | 查询日期 YYYY-MM-DD，默认当天 |

**注意：** 闭市后日 K 线数据更新后才会返回当天数据

### 返回字段
| 字段 | 说明 |
|------|------|
| code | 证券代码 |
| tradeStatus | 交易状态 (1 正常 0 停牌) |
| code_name | 证券名称 |
