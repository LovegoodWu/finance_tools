## 文章简介

### 1. 过去 3 年证券公司的年平均收益率
通过 python 调用 BaoStock 接口获取全市场的证券数据，计算出 2015-2017 年全部证券的年平均收益率，经过排名得出结论：排名前十的证券几乎全部来自深交所。

### 2. 基础知识不简单之净资产收益率
ROE（净资产收益率）是衡量上市公司盈利能力的重要指标。巴菲特曾说：衡量一个公司是否优秀，如果只用一个财务指标，那就是净资产收益率（ROE）。如果一个公司长期的 ROE 保持 15% 以上，基本上可以算作一个好公司。

### 3. 基础知识不简单之市现率
市现率（PCF）是股票价格与每股现金流量的比率，对应公司三大报表之现金流表。

### 4. 基础知识不简单之市盈率
市盈率（P/E ratio）也称"本益比"、"股价收益比率"。通俗地说，市盈率代表你以当前该证券的价格购买该证券后，以当前证券的每股盈余计算，购买成本返回的年数。

### 5. 基础知识不简单之市净率
市净率（PB）是每股股价与每股净资产的比率，是常见的量化因子。

### 6. 大阳线和大阴线
介绍阳线和阴线的基本概念、量化准则，以及阳线阴线的一般应用规律，并计算了单个证券在某条应用规律下的胜出概率。

### 7. 日线 CCI 的计算与展示
CCI 指标（顺势指标）专门测量股价是否属于超买超卖类。常用判断规则："+100 以上为超买区"，"-100 以下为超卖区"。

### 8. Python 实现 RSI 指标的超买和超卖信息提示
RSI（相对强弱指标）是衡量证券自身内在相对强度的指标，能够反映出市场在一定时期内的景气程度。

### 9. Python 实现 KDJ 的超买和超卖信息提示
KDJ 指标（随机指标）是一种相当新颖、实用的技术分析指标，是期货和股票市场上最常用的技术分析工具。

### 10. Python 实现 MACD 金叉和死叉的提示
MACD（平滑异同移动平均线）是常用的技术指标。金叉：DIFF 由下向上突破 DEA，为买入信号；死叉：DIFF 由上向下突破 DEA，为卖出信号。

### 11. Buffett's Alpha 巴菲特的阿尔法
对伯克希尔哈撒韦股票的超额收益进行回归分析，分析市场收益率、规模因子、价值因子、动量因子、对赌 beta 因子和质量因子的影响。
